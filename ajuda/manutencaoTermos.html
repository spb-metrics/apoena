Sistema Apoena
 

Para manter o módulo Termos siga os seguintes passos:
 1 - Para alterar Termos acesse o item Termos->Alterar. A figura abaixo mostra a página de alteração de Termos



2 - Altere os campos com os novos dados selecionando primeiro, uma Ontologia no comboBox "Ontologia" para aparecer seus Termos. Alterações realizadas clique no botão alterar , a mensagem "dados alterados com sucesso" aparecerá. Caso não selecione uma Ontologia e clique no botão alterar a seguinte mensagem irá aparecer "selecione um item"
3 - Para consultar Termos acesse o item Termos->Consultar. A figura abaixo mostra a página de consulta do módulo Termos



4 - A tela mostra um comboBox com Ontologia, escolha uma, e aparecerá uma lista com todos os termos vinculados à Ontologia selecionada.
5 - Para deletar Termos acesse o item Termos->Deletar. A figura abaixo mostra a página de deleção de Termos



6 - A tela mostra um comboBox com Ontologia, escolha uma, e aparecerá uma lista com todos os termos vinculados à Ontologia selecionada. Selecione os Termos que desejar deletar e clique no botão deletar, após a deleção aparecerá a seguinte mensagem "dados deletados com sucesso". Caso você não escolha uma Ontologia ou Termos e clique no botão excluir a seguinte mensagem aparecerá "selecione um item".
7 - Para incluir Termos acesse o item Termos->Incluir. A figura abaixo mostra a página de inclusão de Termos



8 - Pelo menos um campo deve ser preenchido. A página contém um comboBox com Ontologia e campos para preenchimento dos novos Termos. Escolha uma Ontologia e preencha com os Termos, após a inclusão a mensagem "dados incluídos com sucesso aparecerá". Caso não seja escolhida uma Ontologia ou preenchido pelo menos um dos campos para inserção de Termos o sistema emitirá a seguinte mensangem "selecione item"
