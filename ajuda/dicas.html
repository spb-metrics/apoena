Sistema Apoena
 

Apoena é um Software Livre desenvolvido em script shell, java e php. Usa o banco de dados Mysql.

1 - Modelo de dados

2 - Dicas para Programação

Processamento Batch

1) no diretório rss existe um script recolhe que incia tres processos paralelos para ler faixas de rss da base de dados, padroniza e preparar os arquivos .
2) nos diretório transacao e processamento estão os programas java que lêem os arquivos preparados no item anterior, tornando a informação desestruturada em informação estruturada, dentro do banco de dados Mysql.

Processamento Online - em 3 camadas

1) Camada de apresentação - arquivos tpl dentro do diretório templates
2) Camada de persistência - arquivos php dentro do diretório apoena
3) Camada de acesso ao bancos de dados - arquivos ocultos .xxx.php no diretório apoena
