Sistema Apoena
 

Para manter o módulo Fonte siga os seguintes passos:
 1 - Para alterar a Fontes acesse o item Fonte->Alterar. A figura abaixo mostra a página de alteração das Fontes



2 - Altere os campos com os novos dados selecionando, primeiro uma Fonte no comboBox "Fonte" para aparecer seus dados. Alterações realizadas clique no botão alterar , a mensagem "dados alterados com sucesso" aparecerá. Caso não selecione uma Fonte e clique no botão alterar a seguinte mensagem irá aparecer "selecione um item"
3 - Para consultar Fonte acesse o item Fonte->Consultar. A figura abaixo mostra a página de consulta do módulo Fonte



4 - A tela mostra uma lista de Fonte ordenar pelo nome.
5 - Para deletar Fonte acesse o item Fonte->Deletar. A figura abaixo mostra a página de deleção das Fontes



6 - A tela mostra uma lista com o nomes das Fontes, escolha as Fontes que desejar deletar e, clique no botão "deletar", após a deleção aparecerá a seguinte mensagem "dados deletados com sucesso". Caso você não escolha uma Fonte e clique no botão deletar a seguinte mensagem aparecerá "selecione um item".
7 - Para incluir Fontes acesse o item Fonte->Incluir. A figura abaixo mostra a página de inclusão de Fontes



8 - Todos os campos devem ser preenchidos. A página contém um comboBox com os Tipos de Fontes e campo texto para preenchimento das Fontes. Escolha um Tipo de Fonte e preencha o campo com o nome da nova Fonte, depois clique no botão "incluir", após a inclusão a mensagem "dados incluídos com sucesso aparecerá". Caso não seja escolhido um Tipo de Fonte ou preenchido o campo do nome da Fonte o sistema emitirá a seguinte mensangem "selecione item"
